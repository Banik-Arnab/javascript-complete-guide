1.1 What Is JavaScript?
JavaScript is the programming language of the web. Every interactive thing you see in a browser — a like
button, a live search box, a form that checks your email address — is powered by JavaScript. With
Node.js, the same language also runs on servers, which is why one language can take you from a simple
web page all the way to a full-stack application.
A program is nothing more than a list of instructions the computer executes from top to bottom. Your
job as a programmer is to break a problem into steps small enough that each one can be written as an
instruction.
