2.1 Variables: Boxes with Labels

A variable is a named container that stores a value. Modern JavaScript declares variables with two
keywords:
• let — for values that may change later.
• const — for values that must never be reassigned. Prefer const by default; switch to let only when
you know the value will change.

2.2 Naming Rules
• Use camelCase: totalPrice, firstName, isLoggedIn.
• Names may contain letters, digits, _ and $, but cannot start with a digit.
• Make names meaningful. s tells you nothing; studentName tells you everything.
